Integrate EDD with the Vero email marketing platform

== Description ==

[Vero](http://www.getvero.com/) is an advanced email platform that offers automation, real time event tracking, and powerful segmentation tools for your business.

This extension allows store owners to add customer data to Vero when purchases are made through Easy Digital Downloads. This then allows store admins to utilize the powerful automation tools in Vero to send follow-up emails, marketing material, and more.

== Installation ==

1. Unzip the downloaded zip file
2. Upload the included folder to '/wp-content/plugins' directory of your WordPress installation
3. Activate the plugin via the WordPress Plugins page
